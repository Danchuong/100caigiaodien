# Docs

Đây là thư viện phụ. File cần mở hằng ngày nằm ở root:

```text
README.md
100-design-board.md
yeucau.md
100-home-styles.md
```

## Active Support Docs

| Path | Vai trò |
| --- | --- |
| `design-research/brief-template.md` | Template brief cho từng `des-N`. |
| `design-research/des-briefs/` | Brief thực thi từng design. |
| `design-research/refero-source-pool.md` | Pool Refero đã lọc. |
| `getdesign/` | Source getdesign local. |
| `refero/` | Cache Refero local. |

## Archive

`archive/` chứa research và ghi chú cũ. Chỉ đọc khi cần tra cứu lịch sử, không dùng để tracking.

Nếu archive mâu thuẫn với `yeucau.md`, `100-home-styles.md` hoặc `100-design-board.md`, bỏ archive.
