# 100 style home

File này là **nguồn chính để đánh dấu tiến độ 100 style**.

Quy ước đơn giản:

- `Style 1` làm trong theme `des-1`.
- `Style 2` làm trong theme `des-2`.
- ...
- `Style 100` làm trong theme `des-100`.

Không dùng thêm mã `H001`, `S001` hay registry khác.
