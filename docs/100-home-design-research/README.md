# Research cho 100 giao diện home

Folder này chỉ chứa tài liệu nghiên cứu phụ.

File chính để đánh dấu style đã làm là:

```text
100-home-styles.md
```

## Các file nghiên cứu

| File | Vai trò |
| --- | --- |
| `01-real-site-inspiration.md` | Pattern layout từ website game/news/store/editorial thật. |
| `02-visual-language-systems.md` | Hướng visual language, palette, typography, card, background. |
| `03-anti-duplicate-taxonomy.md` | Framework chống trùng khi cần review sâu. |
| `04-first-20-concepts.md` | Ý tưởng concept cho 20 mẫu đầu. |
| `05-quality-review-and-rules.md` | Checklist review visual/kỹ thuật. |

Không dùng folder này để đánh dấu tiến độ.
